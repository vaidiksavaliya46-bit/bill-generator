## ✨ Features

### 🎨 Templates
| Template  | Style                                      |
|-----------|--------------------------------------------|
| Modern    | Gradient header with custom accent color   |
| Classic   | Dark navy with gold accent border          |
| Minimal   | Clean black & white — ultra professional   |
| Elegant   | White with gold sidebar — premium feel     |

### 📋 Invoice Fields
- ✅ Company name, tagline, address, GST number
- ✅ Custom Bill Title — `INVOICE` / `RECEIPT` / `QUOTATION` / anything
- ✅ Invoice number, issue date, due date
- ✅ Customer name, address, GST number
- ✅ Payment status stamp — **Paid / Pending / Unpaid**

### 🛒 Line Items
- ✅ Add unlimited items with name, description, quantity, rate
- ✅ Auto-calculates item totals in real time
- ✅ Remove any item instantly

### 💰 Smart Calculations (all live)
```
Subtotal
  − Discount %
  = After Discount
  + GST / Tax %
  + Shipping / Extra charge
  = GRAND TOTAL
```

### 💱 5 Currencies Supported
| Symbol | Currency            |
|--------|---------------------|
| ₹      | INR — Indian Rupee  |
| $      | USD — US Dollar     |
| €      | EUR — Euro          |
| £      | GBP — British Pound |
| ¥      | JPY — Japanese Yen  |

### 🖨️ Print & PDF Export
- Print Panel with **4 paper sizes**:
  - 📄 **A4** — 210 × 297 mm (International standard)
  - 📃 **Letter** — 8.5 × 11 in (US standard)
  - 🗒️ **A5** — 148 × 210 mm (Compact / Half page)
  - 🧾 **Thermal / POS** — 80 mm receipt roll
- Sidebar & toolbar auto-hide during print
- All colors print correctly (`print-color-adjust: exact`)
- Save as PDF via browser print dialog → **Destination → Save as PDF**

### 💾 Save & Load
- Export any bill as a `.json` file
- Reload it anytime — all fields, items, template, color restore perfectly

### 🔍 Zoom Controls
- Zoom in / out / reset on the live preview — `40%` to `160%`

### 🎨 Customization
- Live accent color picker — changes header, labels, grand total
- Notes & Terms section
- Custom footer text

## 🚀 Getting Started

### Option 1 — Open directly
```
Just open bill-generator.html in any modern browser.
No installation. No server. No npm. Nothing.
```

### Option 2 — Clone & run
```bash
git clone https://github.com/yourusername/billforge-pro.git
cd billforge-pro
open bill-generator.html
```

### Option 3 — Host on GitHub Pages
```bash
# Push to GitHub → Settings → Pages → Deploy from main branch
# Your bill generator goes live at:
# https://yourusername.github.io/billforge-pro/
```
